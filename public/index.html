The Task

Create an App that takes a number [0 - 999,999,999,999] and converts it into its spoken equivalent. For example, 167 would convert to "one hundred and sixty seven".

1. Starting the App

Using the type="text" parameter in the input field restricts the input to numbers only. Then we add a keyup event to run our program every time the value in the input changes and focus the cursor in the box.
